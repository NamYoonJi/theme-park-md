theme-park-md

AI-generated pages all look the same.

A collection of self-contained design-system specs — one markdown file per theme. Hand them to your assistant and the output stops being generic.

Usage

Two steps to a deliberate page.

  1. Clone the repo

    Pull the collection so the specs sit beside your project: git clone https://github.com/NamYoonJi/theme-park-md

  2. Point your assistant at a spec

    Reference one file and tell it to follow the spec exactly — including the Don't section. That single instruction does the work.

Tell your assistant

Build the page as a single HTML file. Follow ~/theme-park.md/themes/midnight.md exactly, including the Don't section.
